# Strata Platform Hub

Landing page for the Strata distributed infrastructure platform — MonoFS, Guardian, Doctor, Agent, LB-Edge, and more.

## Quick Start

```bash
hugo server -D
```

Open http://localhost:1313 in your browser.

## Build

```bash
hugo --minify
```

Output goes to `public/`.
